О программе

Frontend-направление рассчитано на начинающих разработчиков и тех, кто приходит в сферу с нуля. Если вы любите решать задачи креативно, хотите работать над клиентской и визуальной частью сайтов или посадочных страниц, понимаете, что такое компьютерная верстка, мы рекомендуем это направление. Программа даст толчок к продвинутому уровню, а менторы всегда будут рядом, чтобы помочь освоиться и получить все возможные необходимые знания.

Мы будем изучать один из самых популярных языков программирования JavaScript, React. js, работать с Git. Курс проиллюстрирован примерами из реальной жизни, одобрен израильскими программистами, поэтому соответствует последним стандартам рынка.

Студенты пройдут программу, которая включает 5 модулей, посетят воркшопы, в том числе по основам BackEnd с использованием NodeJS, Mobile App, Business Inteligent. Научим решать сложные задачи индивидуально и в группах. Все выпускники получат шанс показать приобретенные навыки на основе своих проектов. Студенты приобретут опыт и научатася презетовать себя и свою работу.

Вступительные требования

Возраст от 18 лет

Знание английского не ниже уровня Pre-Intermediate

Наличие персонального ноутбука или компьютера

Готовность к интенсивному обучению

По окончанию обучения будете

  1. Знать популярный язык программирования JavaScript и уметь работать с ним на продуктивном уровне
  2. Понимать логику работы клинет-серверной архитектуры в контексте написания реальных приложений
  3. Уметь работать с фреймворком React.js - создание приложений с нуля, написане тестов
  4. Работать с системой контроля версий GIT
  5. Разрабатывать веб-интерфейсы и веб-приложения
  6. Читать чужой код с пониманием того, как он работает
  7. Владеть навыками soft-skills и быть готовым к прохождению собеседований, уметь презентовать себя и свой продукт

Программа

Модуль 1.Основы HTML, CSS

HTML – средство логической разметки веб-страницы. Используя структуру разметки, можно вносить данные на страницу. В процессе первого модуля студенты научатся описывать эту структуру, используя HTML-теги (специальные маркеры, которые считывает браузер).

Также в первом модуле студенты будут изучать основы CSS, это нужно для того, чтобы они понимали, как сообщить браузеру правильное отображение HTML-тегов и добиться красивой отрисовки страницы согласно макету.

Обучение языку включает в себя:

  • основные инструменты разработчика
  • селекторы CSS, их вес, и как он вычисляется
  • оформление элементов
  • управление размером и положением блоков на странице
  • блочная модель элементов: границы, отступы
  • работу со стандартами CSS3 и HTML5
  • базовую анимацию
  • основы работы с системой контроля версий

Модуль 2.Основы JavaScript

HTML задает разметку сайта, CSS отвечает за внешний вид, а JavaScript приводит всё в действие.

С помощью JavaScript-кода программист определяет, как страница отреагирует на действия пользователя. Модуль познакомит студентов с синтаксисом языка, научит им пользоваться и позволит написать первые интерактивные страницы.

Студенты получат опыт в таких направлениях:

  • типы данных, их отличия и области применения;
  • преобразование типов
  • переменные и правила их именования
  • условия, операторы, массивы, циклы, функции
  • области видимости переменных
  • контекст выполнения функций
  • управляющие конструкции
  • объектная модель
  • объектная модель окна
  • Localstorage/Sessionstorage
  • браузерные события, отклик на нажатия мыши и клавиатурных клавиш
  • подключение и использование библиотеки jQuery
  • основы работы с jQuery плагинами

Также студенты ознакомятся с порядком обработки событий (event loop), функциями обратного вызова (callback). По ходу курса студенты научатся работе с консолью разработчика, отлаживать (debug) и проверять код на предмет ошибок.

После окончания первых двух модулей студенты фронтенд направления смогут создать веб-страницу с необходимой разметкой и интерактивным поведением.

Модуль 3.Advanced HTML\CSS

Любая веб-страница должна быть удобной для использования и выглядеть на всех устройствах презентабельно и читабельно. Для того, чтобы это реализовать, в программе предусмотрен модуль продвинутых знаний верстки.

Студенты получат знания в направлениях:

  • препроцессоры SCSS, SASS
  • CSS методология БЕМ
  • верстка с пиксельной точностью
  • адаптивная верстка
  • Mobile first верстка
  • сборщики проектов, Gulp и его модули
  • продвинутое использование системы контроля версий Git

Модуль 4.Advanced JavaScript

Модуль поможет углубиться в изучение основ объектно-ориентированного программирования (ООП), базовых шаблонов проектирования, основ функционального программирования, возможностей стандарта ES6\ ES7 (ECMAScript), MVC архитектуры.

Также в этом модуле будут рассмотрены:

  • деструктуризация
  • передача контекста выполнения – методы call, apply, bind
  • замыкания
  • работа «сборщика мусора»
  • обработка ошибок
  • основы объектно ориентированного программирования
  • прототипирование
  • классы стандарта EcmaScript6
  • Drag&Drop
  • основы клиент-серверной архитектуры
  • основы регулярных выражений
  • POST, PUT, DELETE запросы
  • HTTP протокол
  • cookies
  • Async/Await

Студентов ждёт знакомство со стандартами Web Workers API, WebSocket, JSON, AJAX, XMLHttpRequest, Promise, Single Page Application, общая информация про RESTful API.

Модуль построен с учетом последних стандартов в индустрии.

Модуль 5.Основы работы с библиотекой React.js

Изучение построения сложных веб-приложений с помощью React.js. Студенты узнают про принципиально новый и инновационный подход в формировании страниц сайтов и приложений.

Модуль покрывает следующие темы:

  • верстка JSX
  • виртуальный DOM
  • components
  • работа с Props/State компонента
  • React Router
  • React Forms
  • React hooks
  • Redux
  • работа с ImmutableJS в React/Redux приложении.

Дополнительные навыки

Курс предусматривает обязательное изучение системы контроля версий Git, благодаря которой можно грамотно построить работу в команде.

Будут раскрыты следующие темы:

  • работа с Git Bash
  • workflow от создания репозитория до его отправки на удаленный сервер
  • работа с ветками
  • работа с конфигурационными файлами.

Воркшопы

На протяжении всех модулей студенты смогут посещать воркшопы и участвовать в вебинарах от международных специалистов в области ИТ.

Воркшоп "Business Intelligence"

Студенты получат понимание о сфере "Business Intelligence"(BI) и аналитике данных, узнают о развитии бизнес-анализа и его использовании. Основная тема: технологии, идеи для реализации хранилищ данных, визуализация данных и будущее направления BI.

Чему научатся студенты:

  • Сформируют понимание о том, что такое бизнес-анализ
  • Получат знания о новых инструментах аналитики данных
  • Сгенерируют идеи для реализации хранилища данных
  • Обсудят перспективы изучения этой сферы

Воркшоп "Mobile Development на базе React Native"

React Native — фреймворк, который позволяет использовать один язык программирования и для Android, и для IOS в разработке нативных мобильных приложений.

На воркшопе будут рассмотрены такие темы:

  • React и React Native: главные отличия
  • Настройка окружения
  • Создание билда проекта для Android и IOS
  • Использование специальных инструментов
  • Дебаг приложения

Воркшоп "Node JS"

Node JS - программная платформа, которая позволяет использовать JavaScript в куда более широком спектре применений. Благодаря Node JS стало возможным, без изучения дополнительного BackEnd языка разрабатывать серверную часть приложений.

Охват тем мероприятия:

  • введение в платформу Node.js
  • настройка среды и инструментов для разработки
  • организация приложений, модульный подход
  • работа с консолью (терминалом)
  • создание простого веб-сервера, работа с http-запросами
  • обработка событий (особенности работы Event Loop)
  • фреймворк Express
  • размещение Node.js приложения на облачном сервисе Heroku

Soft skills, технический английский и подготовка к собеседованию

С середины курса студенты параллельно начнут заниматься развитием своих внутренних качеств и готовиться к презентации проектов.

Мы создадим условия полноценных собеседований, максимально приближенные к реальным.

Проработаем резюме каждого студента, поработаем над его умениями презентовать проект и свой бренд.

Финальный проект

Команды студентов будут работать над финальным проектом по ролям, с учетом выполненных задач согласно методологии SCRUM. Идея проекта: интернет-магазин со своим продуктом и уникальным интерактивным интерфейсом. По окончанию финального проекта студентам предстоит его защита и презентация.
